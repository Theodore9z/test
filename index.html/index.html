Qui suis-je !

Je suis Theodore Badiane, 18 ans, passionné d'informatique avec des bases solides en HTML/CSS. Mon ambition : transformer ma passion en carrière dans le développement web.

Avec un Bac Pro Commerce, ma passion croissante pour le développement web guide mes choix.

Ayant maîtrisé les bases du code HTML, je suis déjà en mesure de créer des structures et des contenus pour des sites web. Mon objectif est d'élargir progressivement mon champ de compétences.

Mon ambition ultime est de devenir un développeur web accompli, en apportant des idées novatrices et une exécution impeccable à chaque projet sur lequel je travaillerai
